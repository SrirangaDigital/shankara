------------------------------------------------------------------------

					About The Shankara Granthavali Archive

------------------------------------------------------------------------

This Readme file is applicable to the USB Card Drive version of the archive.

The archive can be viewed on computers with Windows 7, 8, 10, Linux or
macOS operating systems.

Please note that the archive can be used only if the 
original media (USB Card Drive) is inserted into the computer.

Usage instructions for Windows
------------------------------------
Open the file "For Windows", the installation will
happen automatically and the archive will launch.
For subsequent usage, the desktop icon "For Windows" could be used.
To uninstall, Control Panel > Programs > Programs and Features > For Windows 1.0.0 (right click) > Uninstall

Usage instructions for Linux
------------------------------------
Copy the file "For Linux" to any location in your
computer and make the file executable.
	a) chmod +x "For Linux"
or	b) Right Click > Properties > Permissions > Allow executing file as program
To launch the archive, open this file.
For subsequent usage, the same file, thus copied, could be used.
To uninstall delete the file, thus copied.

Usage instructions for macOS
------------------------------------
Open the package "For macOS.dmg" and extract the 
contents to Applications folder.
Launch the archive from the Applications menu by clicking on "For macOS"
For subsequent usage, the Applications menu item "For macOS" could be used.
To uninstall delete "For macOS" from Applications.

General troubleshooting instructions
------------------------------------
While trying to read an article, if the screen is empty with a rotating icon, 
please make sure that the media (USB Card Drive) is inserted 
into the computer and is active.

For further queries and a more elaborate FAQ please visit https://sriranga.digital/SG-faqs


1) If you see a single page view when viewing books in a bookreader then please click on the two page mode at the bottom for better experience.
2) If you face any problem while viewing books navigating from search results page then do the following

	i) go to the volumes page
   ii) click on corresponding volume which will take you to the TOC for that volume
  iii) click on any of the TOC entry which will open the book
   iv) on the top right edge of the bookreader there is a search text field, enter the search term there and then hit enter button on the keyboard
    v) if the search term is found then blue colored droplets will be shown on the progress bar near the bottom of the bookreader. 
   vi) please click on those droplets to go the corresponding page. Search term will be highlighted on those pages.
